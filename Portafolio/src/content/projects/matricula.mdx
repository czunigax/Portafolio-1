## 📚 Descripción  
Este sistema simula el proceso de matrícula en la UNAH, permitiendo a los estudiantes gestionar su inscripción de asignaturas y a los docentes administrar la subida de notas.  

### 🛠️ Tecnologías usadas  
- **Java** para la lógica de negocio  
- **Spring Boot** para la API  
- **React.js** para la interfaz de usuario  
- **MySQL** para la base de datos  

### 🚀 Características  
✅ Inscripción de asignaturas según fechas definidas  
✅ Listado de clases disponibles con cupos actualizados  
✅ Subida y gestión de notas por los docentes
