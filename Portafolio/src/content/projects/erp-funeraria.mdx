## ⚰️ Descripción  
Este sistema ERP está diseñado para la administración de funerarias, proporcionando herramientas de gestión para clientes, empleados y análisis de negocio.  

### 🛠️ Tecnologías usadas  
- **Laravel y PHP** para el backend  
- **Vue.js** para la interfaz de usuario  
- **MySQL** para la base de datos  

### 🚀 Características  
✅ Gestión de clientes y servicios funerarios  
✅ Administración de empleados y tareas asignadas  
✅ Reportes financieros y Business Intelligence
