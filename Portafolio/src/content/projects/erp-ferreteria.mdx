## 🏗️ Descripción  
Este sistema ERP está diseñado para optimizar el control de inventario y la gestión de ventas en ferreterías, asegurando un uso eficiente de los recursos.  

### 🛠️ Tecnologías usadas  
- **Django** y **Python** para el backend  
- **Vue.js** para la interfaz de usuario  
- **PostgreSQL** para la base de datos  

### 🚀 Características  
✅ Gestión de inventario en tiempo real  
✅ Facturación automatizada  
✅ Reportes detallados de ventas y stock
